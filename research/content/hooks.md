## Video Hook Formulas

### The Demo Hook
```
[3s] Show coffee being poured
[2s] Add FUSE powder
[3s] Watch it vanish
[2s] Sip and nod
[5s] Text overlay: "Britain's first coffee creatine"
```

### The Blind Test Hook
```
[3s] Two identical coffees
[3s] "One has creatine. Can you tell which?"
[10s] Tester tries both
[3s] Reveal: "Neither could you"
```

### The Transformation Hook
```
[5s] Old routine: shaker, powder, mess
[5s] New routine: just add to coffee
[5s] "Same results. Zero hassle."
```

### The Science Hook
```
[3s] "500 studies. 1 supplement. 2 seconds."
[5s] Quick stat visuals
[5s] Demo the dissolve
[2s] "This is FUSE."
```

## Platform-Specific Adaptations

### TikTok
- Start with motion (pouring coffee)
- Text on screen immediately
- Face in first 2 seconds if possible
- Sound on: ASMR coffee sounds, calm voice
- 15-30 seconds max for initial tests

### X (Twitter)
- Text-first, video optional
- Strong opening line
- Thread format for story/science
- Poll integration
- Founder voice authentic

### Facebook
- Slightly longer acceptable
- More context in captions
- Target: 35-55 coffee enthusiasts
- Groups: biohacking, fitness over 30

### Instagram Reels
- Aesthetic priority
- Coffee ritual beauty shots
- Minimal text on screen
- Carousels for education

## Hook Testing Protocol

### Phase 1: Raw Test (No Production)
1. Film on iPhone
2. Natural lighting (kitchen, cafe)
3. Post with 3 different hooks
4. Track: views, completion rate, comments

### Phase 2: Refinement
1. Top hook gets production treatment
2. Better audio, lighting, editing
3. A/B test variations
4. Track: saves, shares, DMs

### Phase 3: Scale
1. Winning hook → multiple platforms
2. Adapt length/format per platform
3. Track: waitlist conversions
4. Loop learnings back
