## Content Production Workflow

### Monday: Plan
- Review previous week's metrics
- Identify top-performing content
- Plan week's content angles
- Update research notes

### Tuesday-Thursday: Create
- Film 2-3 raw videos
- Write 3-4 text posts
- Design 1-2 images
- Schedule posts

### Friday: Distribute
- Post to all platforms
- Engage with early comments
- Track initial performance

### Weekend: Engage + Learn
- Respond to all comments/DMs
- Monitor sentiment
- Document insights
- Update pain points file

## Content Templates

### Poll Template (X)
```
[Question in clear language]

◯ Option A
◯ Option B
◯ Option C
◯ Option D

RT = I want to see results
```

### Thread Template (X)
```
[Hook - stop the scroll]

[Problem - make them feel it]

[Insight - surprise them]

[Solution - introduce FUSE]

[Proof - why it works]

[CTA - what to do next]
```

### Video Template (TikTok)
```
0-3s: Visual hook (coffee pour, powder, action)
3-8s: Problem or question
8-20s: Demonstration
20-30s: Result + CTA
```

### Story Template (All)
```
Frame 1: Hook question
Frame 2: Behind-scenes moment
Frame 3: Insight or stat
Frame 4: CTA (poll, link, or question)
```

- Sounds matter: use trending or coffee ASMR
- Hook in first 1 second

- Groups > Page for early traction
- Target: 30-50 age range, fitness interest
- Longer captions OK
